# Wireframe Visual Constraints

These constraints are NON-NEGOTIABLE. Every wireframe generated by `/sh:ux-design` MUST follow all rules below. The purpose is to prevent premature visual design and keep focus on structure and flow.

## Color

- **Lo-fi:** Grayscale only — `#333`, `#666`, `#999`, `#ccc`, `#e5e5e5`, `#f5f5f5`, `#fff`
- **Mid-fi:** Grayscale + ONE brand accent color for primary actions only
- **Never:** Gradients, shadows, glows, opacity effects

## Typography

- **Font:** `system-ui, -apple-system, sans-serif` — no custom fonts, no web fonts
- **Sizes:** 3 tiers max — heading (24px), body (16px), caption (12px)
- **Weight:** Regular and bold only — no light, semibold, or extra-bold
- **Never:** Decorative fonts, letter-spacing tricks, text gradients

## Layout

- **Base width:** 375px (mobile-first)
- **Max width:** 768px (tablet)
- **Spacing:** 8px grid (8, 16, 24, 32, 48)
- **Structure:** Single column. No multi-column layouts, no sidebars, no grids
- **Never:** Absolute positioning, z-index layering, overlapping elements

## Components

- **Buttons:** Solid rectangles with border + label. Primary = dark fill. Secondary = outline only
- **Inputs:** Simple bordered rectangles with placeholder text
- **Images:** Gray rectangle with centered label `[Image 16:9]` or `[Avatar]`
- **Icons:** Square placeholder with label `[icon: search]` — no actual icon fonts or SVGs
- **Cards:** Bordered rectangle with padding — no shadows, no rounded corners beyond 4px
- **Navigation:** Simple text links or tab bar — no hamburger menus, no drawer animations

## Interaction

- **Primary action:** Exactly ONE per screen — visually distinct (dark button)
- **Secondary actions:** Outline buttons or text links
- **Progress indicator:** Step counter visible on every screen (`Step 2 of 6`)
- **Back navigation:** Always present except on first screen
- **Skip:** Explicit skip links where flow allows optional steps

## Content

- **Headings:** Real copy that describes the step ("Connect your calendar")
- **Body text:** Real or near-real copy — NOT lorem ipsum
- **Labels:** Descriptive field labels ("Email address" not "Field 1")
- **CTAs:** Real action verbs ("Continue", "Skip for now", "Connect Google")
- **Never:** Lorem ipsum, placeholder gibberish, emoji, marketing fluff

## Anti-patterns (immediate fix if detected)

| Violation | Fix |
|-----------|-----|
| Custom font loaded | Remove — use system-ui |
| Shadow on any element | Remove — borders only |
| More than 1 accent color | Remove extras — grayscale + 1 max |
| Multi-column layout | Collapse to single column |
| Real images or icons | Replace with labeled gray rectangles |
| No progress indicator | Add step counter |
| No back button | Add back navigation |
| Two primary CTAs on one screen | Split into two screens or demote one to secondary |
| Lorem ipsum | Replace with real copy |
